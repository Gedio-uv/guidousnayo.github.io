DeutschLernen — A German learning tool I built because the existing ones didn't work for me.

Role
UX Designer + Builder
Built with
Antigravity, Groq API, Unsplash API
Timeline
6 hours
Status
Live · In active use

Context

Living in Leipzig and studying German, I found existing vocabulary apps either too gamified or too rigid. None of them gave me what I actually needed — quick word lookup with visual context, grammar rules I could reference on the go, and a way to practice recognition without committing to a full lesson.

So I built one. DeutschLernen is a mobile-first web app for German learners — designed around real daily use, not engagement metrics.

The Problem

Learning German as an adult living in Germany means needing answers fast — on the street, in a shop, reading a sign. Existing tools require too many taps to get to a simple answer, and none of them adapt to whether you're a beginner or already comfortable with the language.

Design Approach

UX First, AI Second

My focus was entirely on UX — the flows, hierarchy, and information architecture. I delegated UI execution to Antigravity, which handled visual decisions within a system anchored to the German flag palette. That constraint — trusting the tool with UI while owning UX — is exactly how I approach AI-assisted design in product work.

Key UX Decisions

  • Level selector at entry

    The first screen asks Beginner or Advanced. This single decision changes the entire interface language — native language for beginners, full German for advanced. No settings buried in a menu.

  • Search as the core

    Word lookup returns gender, pronunciation, translation, synonyms, example sentences, and a contextual image in one screen. Zero navigation needed for the primary use case.

  • Cards as passive learning

    The flashcard game uses visual recognition, not translation. You see the image, identify the word. Flipping the card confirms or corrects. Low friction, high retention.

  • Grammar as reference, not lesson

    The grammar section is structured as a quick reference for cases, articles, verbs, and word order. Designed to answer a specific question in 10 seconds, not to teach from scratch.

AI Integration

Built for Speed

Groq API powers word lookup, definitions, and example sentence generation instantly.
Unsplash API surfaces contextual images for each word result.
Antigravity was used for UI generation and front-end assembly.

Result: functional product in 6 hours.
